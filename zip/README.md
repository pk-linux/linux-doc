# Zip

## Everything related to zip, unzip, gunzip, tar and other zip tools.

1. To zip a directory

   ```sh
   $ zip -r zip_name.zip /path/to/folder/
   ```

2. To unzip a zip file

    ```sh
    $ unzip file.zip -d destination_folder
    ```

# GZ

1. To unzip .gz file


```sh
$ gunzip name_of_gz_file.gz
```
